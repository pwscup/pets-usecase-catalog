# Task 001: Vite + React + TypeScript プロジェクト初期化

## フェーズ
Phase 1: プロジェクト初期化 + 一覧表示（MVP）

## 概要
Vite を使って React + TypeScript のプロジェクトを初期化し、Tailwind CSS、ESLint、Prettier を導入する。

## 作業内容
1. `npm create vite@latest` で React + TypeScript テンプレートを作成
2. Tailwind CSS v4 のインストールと設定
3. ESLint + Prettier の設定
4. ディレクトリ構成の骨格作成（src/types, src/hooks, src/lib, src/components, src/pages, src/context）
5. `vite.config.ts` に `base` 設定を追加

## 完了条件
- `npm run dev` で開発サーバが起動する
- `npm run build` でビルドが成功する
- Tailwind のユーティリティクラスが適用される
